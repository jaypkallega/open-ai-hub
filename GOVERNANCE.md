## Scoring Rules

### Karma Score (Creation)
| Action | Points |
|---|---|
| PR merged into `experimental/` | +5 |
| PR merged into `prompts/` or `skills/` | +15 |
| Asset promoted to `community-picks/` | +30 |
| Asset gets 10+ 👍 reactions | +10 |
| Asset cited in another's `forked_from` | +5 per citation |
| Asset flagged and removed | −20 |

### Contributor Score (Maintenance)
| Action | Points |
|---|---|
| Reviewing a PR with substantive feedback | +5 |
| Review leads to a merged improvement | +10 |
| Adding `tested_on_models` entry to someone else's asset | +3 |
| Verified `broken_on_models` report | +5 |
| Merging a community fix into your stable asset | +8 |
| Deprecating your own outdated asset responsibly | +5 |

## Tier Requirements
| Tier | Title | Requirements | Privileges |
|---|---|---|---|
| 0 | Contributor | First merged PR | Submit to `experimental/` |
| 1 | Trusted Contributor | 5+ merged PRs, 50+ Karma | Submit directly to `prompts/` or `skills/` |
| 2 | Recognized Author | 150+ Karma + 80+ Contributor Score + peer nomination | 48h review SLA, direct metadata edits |
| 3 | Core Maintainer | Nominated by existing Maintainer | Merge PRs, promote to `community-picks/`, manage deprecations |
